# Student Grade Management System

A Java-based Student Grade Management System developed using Java Swing GUI and core Java programming concepts.

## 📌 Project Overview

This project allows teachers or users to manage student academic records through a simple graphical user interface (GUI).  
The system stores student information, calculates grades and averages, displays reports, and provides class statistics.

The project was developed as part of the Computer Programming (Java) course mini project.

---

# ✨ Features

- Add student records
- Store student ID and name
- Enter marks for 3 subjects
- Calculate:
  - Total marks
  - Average score
  - Letter grade
  - Pass/Fail status
- Display all students
- Search for students by ID
- Find top student
- Display class summary
- GUI interface using Java Swing
- Exception handling for invalid inputs

# 📊 Grading System

| Average Score | Grade | Status |
|---|---|---|
| 90 - 100 | A | Pass |
| 80 - 89 | B | Pass |
| 70 - 79 | C | Pass |
| 60 - 69 | D | Pass |
| Below 60 | F | Fail |
